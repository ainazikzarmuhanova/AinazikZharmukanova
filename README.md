
Линейлік және бинарлық іздеу алгоритмдерін салыстыру (Java)

Кіріспе

Іздеу алгоритмдері – программалауда ең маңызды тақырыптардың бірі. Олар деректер құрылымында (массив, тізім) қажетті элементті табуға мүмкіндік береді. Іздеу әдістерінің тиімділігі уақытқа және деректер көлеміне байланысты болады. Осы жұмыста ең кең таралған екі алгоритм – линейлік іздеу және бинарлық іздеу қарастырылады.


Мақсаты
	•	Линейлік және бинарлық іздеу алгоритмдерін үйрену
	•	Олардың жұмыс істеу принципін түсіну
	•	Екі алгоритмнің жылдамдығын салыстыру
	•	Қай жағдайда қай алгоритм тиімді екенін анықтау
